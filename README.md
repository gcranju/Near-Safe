# Near-Safe

A multisig wallet web app for the NEAR blockchain — propose, review, and execute transactions that require multiple approvals.

## Features

- Connect with a wide range of NEAR wallets (Meteor, Ledger, HOT, Math, Coin98, Ethereum wallets, and more) via NEAR Wallet Selector.
- Create and manage multisig accounts.
- Propose transactions and track them through approval to execution.
- Review pending proposals and transaction details in a clean dashboard.

## Pages

- **Multisig** — overview of a multisig account and its members.
- **New Transaction** — build and submit a proposal.
- **Transactions / Transaction Detail** — track proposals, approvals, and execution.

## Tech

React · TypeScript · Vite · Tailwind · shadcn/ui · [@near-wallet-selector](https://github.com/near/wallet-selector)

## Run locally

```bash
npm install
npm run dev
```
